About hash-calendar

Last updated: February 12, 2026

Project Purpose

hash-calendar is a privacy-first, client-only calendar designed for fast sharing without accounts. It keeps calendar state in the URL hash so you can share a schedule by sharing a link.

Trust Signals

  • No account required.
  • No server-side calendar database.
  • URL hash data is not sent in standard HTTP requests.
  • Optional password-based encrypted links are supported.
  • Open-source project under the MIT license.

This app is intended for convenience and privacy, but shared links are still sensitive. Treat calendar links like private data.
